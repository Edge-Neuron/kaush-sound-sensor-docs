## Frequently Asked Questions

### Hardware Questions
**Q: What's the maximum input voltage?**
A: The sensor operates safely from 4V to 12V DC.

**Q: Can I use it with 3.3V systems like ESP8266?**
A: Yes, but you'll need a voltage divider for the analog input. See the [Pin Configuration](pin-config.md) guide.

### Software Questions
**Q: Why can't I see my COM port?**
A: Make sure you have the correct USB drivers installed. See the [Troubleshooting](troubleshooting.md) section.

**Q: The library won't compile. What's wrong?**
A: Ensure you have the latest Arduino IDE and the library is properly installed. Check the [Getting Started](getting-started.md) guide.
